# 📅 Date Cycle Finder

https://pawloland.github.io/DateCycleFinder/

## What is this?

**Date Cycle Finder** is a simple web tool that helps you find dates in other years that fall on the **same weekday** as a selected date.

It allows you to:
- Search **past or future years**
- Find dates with the **same weekday**
- Highlight **perfect calendar cycle matches**
- See the **next exact calendar repeat**

---

## Why was this created?

This tool was created mainly to work around limitations in **legacy devices** that only support a **restricted range of years**.

Some older systems:
- Only allow selecting dates within a **fixed year range** (e.g. 2000–2020)
- Still rely on correct **weekday alignment** for proper operation

Because of this, you may need to:
> Pick a *different year* within the allowed range that behaves the same as your desired date.

This tool helps you find such substitute dates by matching:
- The **same day and month**
- The **same weekday**
- Optionally, the **same leap-year structure**

---

## How it works

1. Select your desired date  
2. Choose search direction (past or future)  
3. (Optional) Enable same leap-year pattern  
4. Click **Find Matches**

The tool will:
- List matching dates in other years  
- Show how many years apart they are  
- Highlight full calendar repeats (⭐)  
- Display the next perfect cycle
